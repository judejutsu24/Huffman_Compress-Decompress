Huffman Decompressor & Compressor Programs

Description:
A pair finals project developed by Marco Elijah P. Rodil and Jude S. Tabuzo Jr. for the course: Data Structure.

To compile:
For compiling the Compressor Program: 
	g++ Compressor.cpp src/*.cpp -o Compressor.exe 
For compiling the Decompressor Program:
	g++ Decompressor.cpp src/*.cpp -o Decompressor.exe
